# Vehicle Ads Dashboard

Esta aplicación web fue desarrollada con Streamlit y permite explorar un conjunto de datos
de anuncios de venta de coches en Estados Unidos.

## Funcionalidad
Visualización interactiva de un histograma del kilometraje de los vehículos.
Gráfico de dispersión que muestra la relación entre el precio y el kilometraje.
Interfaz sencilla e interactiva mediante casillas de verificación.

## Tecnologías utilizadas
Python
Pandas
Plotly 
Streamlit

## Estructura del proyecto
app.py: aplicación web principal.
vehicles_us.csv: conjunto de datos utilizado.
notebooks/EDA.ipynb: análisis exploratorio de datos.
requirements.txt: dependencias del proyecto.
